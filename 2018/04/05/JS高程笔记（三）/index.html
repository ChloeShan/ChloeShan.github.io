面向对象的程序设计

理解对象

属性类型

  • 数据属性
    • [[Configurable]]，默认true，能否delete删除属性，修改属性，改成false后，就不能改回true，除了writable以外都不能修改
    • [[Enumerable]]，默认true，能否for-in循环返回属性
    • [[Writable]]，默认true，能否修改属性的值
    • [[Value]]，默认undefined，属性的数据值
    • Object.defineProperty(对象，属性名，描述符对象(value,writable,configurable,enumerable))，修改属性的默认特性，若创建新属性，除了value，默认值都为false
  • 访问器属性
    不包含数据值
    • [[Configurable]]，同上
    • [[Enumerable]]，同上
    • [[Get]]，读取属性时调用的函数，默认undefined
    • [[Set]]，写入属性时调用的函数，默认undefined
    • Object.defineProperty(对象，属性名，描述符对象(configurable,enumerable，get,set))

定义多个属性

object.defineProperties();
var book = {};
object.defineProperties(book,{
    _year:{
        writable: true,
        value: 2004}
})

读取属性的特性

object.getOwnPropertyDescriptor();

创建对象

工厂模式

用函数来封装以特定接口创建对象的细节

function a(name,age){
    var o = new Object();
    o.name = name;
    o.age = age;
    return o;
}

构造函数模式

function A(name ,age){
    this.name = name;
    this.age = age;
    this.say = function(){
        alert(this.name);
    }//每创建一次实例，都会创建一个函数对象
    //this.say = say;
}
//function say(){
//    alert(this.name);
//}
//这样无封装性
var a = new A("a", 10);
//创建一个新对象
//this指向这个新对象
//执行代码，为新对象添加属性
//返回新对象
a instanceof A;//true

原型模式

object.getPrototypeOf(a);//返回a的原型模型
//现在实例中搜索某属性，若没有，则在原型中寻找
