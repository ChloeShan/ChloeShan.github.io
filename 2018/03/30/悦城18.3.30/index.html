I/O流

File类（import java.io.*)

  • 表示文件，文件夹
  • 构造函数
    File(String path)
    File(String path, String filename)
    File(File file, String filename)
    “/“或”\\”转义字符
    • 绝对路径
    • 相对路径，工程文件夹下开始
  • 方法
    getName() 文件名
    getParent() 父目录名
    exists() 判断文件是否存在
    renameTo() 文件改名（移动，更改路径）
    delete() 删除文件
    isFile() 是否为文件
    isDirectory() 是否为文件夹（不能获得文件夹的大小）
    getPath() 返回相对路径
    getAbsoulutePath() 返回绝对路径
    lastModified() 最后修改时间ms
    leng() 文件大小（字节）
    list() 字符串数组 文件名
    listFiles() 数组内file对象 向下遍历一层文件
    如何一层一层遍历？

    流

  • 输入输出
  • 字节 字符（2字节）流，底层以字节的形式实现
    • 抽象类（字节流）
      • InputStream（ByteArrayInputStream()）
        read()
        available() 还剩多少字节可以读
        skip() 跳过几个字节
        mark() 做标记（一个）
        reset() 跳到标记 若没有标记，则回到开头
        close() 关闭
      • OutputStream
        write()
        flush() 强制输出缓冲区的字节
  • Reader
    • FileReader
    • BufferedReader
      • readLine() 可以读自己java文件
      • close() 先关内层流再关外层流 //br = new BufferedReader(fr);先关fr再关br
  • Write
    • FileWriter
      • 构建函数：路径 append //追加，默认覆盖
    • BufferedWriter
      • write() 输出到缓冲区
      • flush() 缓冲区输出
      • newLine();转行
  • FileInputStream
  • FileOutputStream
  • InputStreamReader
  • DataOutputStream

  • DataInputStream

  • 节点流处理流
