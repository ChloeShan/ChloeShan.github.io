流程控制

if else 嵌套

三种循环、break、continue、嵌套

无goto，其为保留字

switch(int byte short char string){
case 常量、编码为0、会自动检测范围（byte）
default 优先级低于 case

}

类与对象

  • 对象是类的具体实例
  • 属性（特征）、类方法（所能完成的操作）
  • 定义，返回值兼容
  • 重载 函数名称相同，参数不同（类型，个数，参数顺序），跟返回值无关，根据参数选择最合适的方法
  • 构造方法 可以重载
  • this this.属性 this.方法 可以调用构造方法 this(参数)(放在首行，不能嵌套调用)
  • 方法执行到return会立即停止，返回到上一层，只能返回一个值

class 类名{
属性 = 默认值 （0、‘0’、null)；
返回值类型 方法名(参数列表){}；
}

main{
类名 对象名 = new 类名();
对象名.属性（访问属性）
对象名.类方法名()（访问方法）

}

类的继承


单继承
子类拥有父类的属性和方法，在其基础上可以拓展出自己的属性和方法

  • 继承的概念和实现方法 extends
  • 子类对象的构建过程、子类构造方法与父类构造方法关系——自动调用、必须定义、super关键字（父类构造方法）
  • 方法的覆盖(重写，名 参数类型 参数个数 都相同)及相关关键字
    1. 就近原则
    2. super可以找到被屏蔽，重写
  • 引用的概念、父类的引用指向子类对象
    1. 调用子类的方法（看实例）
    2. 调用父类的属性（看引用）
    3. 通过方法调用属性（该方法所在类的属性）
    4. 可以父类数组存放不同子类对象

先创建父类再创建子类

父类有参构造，需要在子类构造方法中手动调用父类构造方法super()，且写在第一行

final 属性 意味着不能第二次赋值 在类内
final class 不能派生子类
